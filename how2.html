XDVPN HOW TO

Debian and related distros

  • Run `apt get install openvpn`
  • Download your xdvpn config file
  • Run `sudo openvpn --config /path/to/the/xdvpn/config/file.ovpn`
  • Connected

CentOS and related distros

  • Download and install the correct version of epel6 or epel7, respectivly
  • Run `yum update`
  • Run `yum install openvpn easy-rsa`
  • Download your xdvpn config file
  • Run `sudo openvpn --config /path/to/the/xdvpn/config/file.ovpn`
  • Connected

Windows

  • Download the Windows OpenVPN client from here
  • Download your xdvpn config file
  • Move it to C:\Program Files\OpenVPN\config
  • Run OpenVPN as administrater from the start menu
  • Click the OpenVPN status bar icon, and select "Connect"
